🧠 Multimedia Techniques

This page explains how the project uses media elements and multimedia processing techniques.

🖼️

Image Compression Used

The vocabulary images were first downloaded in PNG format. Because PNG files had large sizes, they were converted into WebP format before being used in the system.

PNG Images → WebP Images
Large File Size → Smaller File Size
Faster Website Loading
🗜️

Compression

Images can be compressed using JPG, PNG, or WebP format. Audio pronunciation files can be compressed using MP3 format.

PNG / WAV → WebP / MP3
🔊

Signal Processing

Audio is processed digitally through speech generation or recording, sampling, encoding, compression, and playback through the browser.

Voice → Sampling → Encoding → Playback
🔐

Data Security

Student score and progress can be protected using input validation, local storage, session control, access control, and password hashing if login is added.

Input → Validation → Storage → Protection
🎤

Audio Signal Processing

The pronunciation feature uses digital audio processing. When a word is spoken, the audio signal passes through several stages before being played to the user.

Voice Input
↓
Sampling
↓
Encoding
↓
Compression
↓
Audio Playback
🔊

Pronunciation Audio

The system uses browser speech synthesis technology to convert text into audio pronunciation. Students can listen to the correct pronunciation of each English word before attempting quizzes.

Text → Speech Synthesis → Audio Output
🔐

Data Security

User input is validated before processing. Student progress can be stored securely using local storage. If the system is expanded, password hashing and session management can be implemented.

User Input ↓ Validation ↓ Secure Storage ↓ Protected Data
